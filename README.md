# 💌 表白程序（超级英雄漫画冒险主题）

一页可部署的静态情书 / 邀请函，采用**原创英雄漫画风**界面：给**老搭档**的「重聚行动简报」、读档续章、霓虹夜色与红金蓝配色，适合微信内置浏览器打开分享。

> GitHub 仓库：**[love-page](https://github.com/zzrwdcr/love-page)**

## ✨ 功能特点

- **微信优化**：完美适配微信内置浏览器
- **精美动画**：飘落的爱心背景 + 烟花庆祝效果
- **打字机效果**：表白文字逐行显示，营造氛围
- **互动按钮**："不愿意"按钮会逃跑，增加趣味性
- **响应式设计**：完美适配各种手机屏幕
- **音效支持**：成功时播放庆祝音效
- **分享卡片**：微信分享时显示精美卡片

## 🚀 快速开始

### 1. 本地预览

直接在浏览器中打开 `index.html` 文件即可预览效果。

### 2. 部署上线（必需）

需要将文件部署到服务器才能通过微信分享链接访问。

#### 方案一：Vercel（推荐，免费）

1. 访问 [vercel.com](https://vercel.com) 注册账号
2. 点击 "Add New..." → "Project"
3. 导入你的代码仓库或直接上传文件
4. 自动部署后会获得一个 HTTPS 链接，如 `https://your-project.vercel.app`
5. 将链接分享到微信即可

#### 方案二：Netlify（免费）

1. 访问 [netlify.com](https://netlify.com) 注册账号
2. 将 `index.html` 拖拽到部署区域
3. 获得部署链接

#### 方案三：GitHub Pages（免费）

1. 将代码上传到 GitHub 仓库
2. 进入仓库 Settings → Pages
3. 选择分支部署
4. 获得 `https://yourname.github.io/repo-name` 链接

#### 方案四：腾讯云开发（国内访问快）

1. 访问 [cloud.tencent.com](https://cloud.tencent.com)
2. 进入云开发 CloudBase
3. 创建静态网站托管
4. 上传 `index.html`

## 📝 自定义修改

### 修改表白文案

打开 `index.html`，找到第 387 行的 `letterText` 变量：

```javascript
const letterText = `从第一次见到你的那一刻起，
        我就知道，你和其他人不一样。
        ...
        你愿意，让我陪伴你吗？`;
```

将你的表白内容替换进去，支持多行文本。

### 修改分享卡片信息

打开 `index.html`，找到第 11-13 行的 meta 标签：

```html
<meta property="og:title" content="给你的一封信">
<meta property="og:description" content="有一件重要的事想告诉你...">
<meta property="og:image" content="https://...">
```

- `og:title`：分享卡片的标题
- `og:description`：分享卡片的描述
- `og:image`：分享卡片的缩略图 URL（建议 300x300 像素）

### 修改配色主题

在 CSS 部分找到渐变定义（约第 38 行）：

```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

可以修改为其他配色：
- 粉色系：`#ff9a9e` → `#fecfef`
- 红色系：`#ff6b6b` → `#ee5a6f`
- 橙色系：`#f093fb` → `#f5576c`

### 修改问题文字

找到第 218 行：

```html
<h3>你愿意做我的另一半吗？</h3>
```

## 📱 使用步骤

1. **部署**：将 `index.html` 部署到任意静态托管平台
2. **获取链接**：复制部署后的 HTTPS 链接
3. **微信分享**：在微信聊天中粘贴链接发送
4. **对方打开**：对方点击链接，在微信内置浏览器中查看
5. **表白成功**：对方点击"愿意"后显示庆祝动画

## 🎯 效果预览

程序包含三个页面：

1. **信封页**：显示一个精美的信封，提示用户打开
2. **表白页**：打字机效果逐行显示表白文字，底部有互动按钮
3. **成功页**：点击"愿意"后显示庆祝动画和成功信息

## 🔧 技术说明

- **纯前端**：HTML + CSS + JavaScript，无需后端
- **响应式**：使用 viewport 和 flex 布局适配各种手机
- **Canvas 动画**：背景爱心飘落和烟花效果使用 Canvas 实现
- **微信兼容**：针对微信内置浏览器做了特殊优化

## 💡 提示

- 分享缩略图需要是外网可访问的 HTTPS 图片链接
- 音频播放受浏览器策略限制，需要用户交互后才能播放
- 建议在正式使用前先在手机上测试效果
